# lab-zhcn-leyu

本仓库用于归档和发布多个独立 HTML 页面。这些页面彼此独立，不归属于任何特定域名或网站，仅作为静态资源存档存储在此处。

## 项目简介

`lab-zhcn-leyu` 是一个实验性质的归档仓库，主要存放各类独立 HTML 文件。每个文件代表一个独立的页面，可能包含不同的内容或功能。仓库不针对任何单一主题或目标，仅作为 HTML 页面的集合存储地。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 独立 HTML 页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `pages/` 目录下存放所有独立的 HTML 页面文件。
- `assets/` 目录存放页面可能引用的公共资源，如样式表、脚本或图片。
- 根目录下的 `index.html` 为可选的导航页，用于列出所有归档页面。

## 页面归档说明

- 每个 HTML 文件均为独立完整的页面，可直接在浏览器中打开使用。
- 页面之间不共享状态或数据，彼此无依赖关系。
- 归档的页面可能来自不同时期、不同用途，内容风格各异。
- 部分页面可能引用 `assets/` 目录下的资源，请确保文件路径正确。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改 HTML 页面。
- 不建议直接修改已归档的页面，除非有明确的修复需求。
- 如需添加新页面，请将 HTML 文件放入 `pages/` 目录，并确保引用资源路径正确。
- 所有提交应保持页面独立性和完整性，避免引入外部依赖或跨域资源。

## 使用方式

1. 克隆本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 或通过 GitHub Pages 等静态托管服务部署后在线访问（需自行配置）。

## 许可

本仓库内容仅供学习、参考及归档使用。具体页面可能包含各自不同的许可声明，请以页面内注明为准。
